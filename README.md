## Getting Started

### Installation
1. Clone the repository:

```bash
https://github.com/MohammadDAlil0/Chatio.git
```
2. Install dependencies:
```bash
cd Chatio
npm install
```
3. Start the server:
```bash
npm start
```
The server should now be running on http://localhost:4000.
